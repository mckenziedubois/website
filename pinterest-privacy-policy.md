# Privacy Policy

**Last updated:** January 2026

This application is an internal automation tool used to publish image content to Pinterest and associate resulting Pinterest Pin URLs with media assets stored in Cloudinary.

---

## Information We Collect

This application does **not** collect personal user data.

The only data processed or stored includes:

- Pinterest OAuth access and refresh tokens (used solely to authenticate API requests)
- Pinterest Pin identifiers and Pinterest Pin URLs created by the application
- Cloudinary asset identifiers associated with those Pins

---

## How We Use Information

The information listed above is used exclusively to:

- Create Pins on Pinterest via the official Pinterest API
- Store resulting Pinterest Pin URLs as structured metadata on Cloudinary-hosted images
- Maintain internal references between Cloudinary assets and Pinterest Pins

---

## Data Sharing

No data is sold, rented, or shared with third parties.

Data is transmitted only to the following services as required for application functionality:

- Pinterest (to create Pins)
- Cloudinary (to store and retrieve asset metadata)

---

## Data Retention

OAuth tokens and Pin references are retained only for as long as necessary to support automated publishing and metadata synchronization. Access can be revoked at any time.
